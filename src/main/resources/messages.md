###### раздел: связь с мессенджером (Макс/тг) connection



connection.main

🔗 Чтобы иметь возможность пользоваться вашим аккаунтом в другом мессенджере потребуется код.



Сгенерировать и ввести его можно по кнопкам ниже:





###### сгенерировать код



connection.makecode

🔒 Ваш код: //



Введите его в другом аккаунте, чтобы связать их и пользоваться как единым аккаунтом.





###### ввести код



connection.getcode

🖋️ Введите код, полученный с другого мессенджера.



connection.wait...

🔗 Подтвердите связь в мессенджере.
