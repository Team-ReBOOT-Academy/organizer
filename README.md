# Кроссплатформенный бэкенд-сервис 
 Кроссплатформенный бэкенд-сервис для управления личными задачами с возможностью интеграции поддерживаемых платформ. Проект разрабатывается командой **Team ReBOOT Academy**.

## 🔥 Стек технологий

Проект построен на современной Java-архитектуре с использованием фреймворка Spring Boot.

* **Язык программирования:** Java 21
* **Основной фреймворк:** Spring Boot 4.0.6
* **Интеграция с Telegram:** TelegramBots Spring Boot Starter (Long Polling)
* **База данных:** PostgreSQL 
* **Сборка:** Maven

## 🚀 Установка и запуск (Локально)

### 1. Подготовка
Для запуска проекта на вашем компьютере должны быть установлены:
* JDK 21
* Maven

### 2. Клонирование репозитория
Откройте терминал и выполните команды:

```bash
git clone [https://github.com/Team-ReBOOT-Academy/organizer.git](https://github.com/Team-ReBOOT-Academy/organizer.git)
cd organizer
```

### 3. Настройка конфигурации (Ключи бота)
Для работы приложения необходим токен Telegram-бота (его можно получить у @BotFather).

Создайте конфигурационный файл для локального запуска или используйте переменные окружения, чтобы передать токен приложению.

> ⚠️ Внимание: Никогда не отправляйте (не коммитьте) реальные токены и пароли от баз данных в открытый репозиторий!

### 4. Сборка проекта
Соберите проект с помощью Maven:

```bash
mvn clean package
```

### 5. Запуск приложения
Запустить проект можно двумя способами:

Способ А: Через плагин Spring Boot:

```bash
mvn spring-boot:run
```

Способ Б: Через скомпилированный JAR-файл:

```bash
java -jar target/tg-organizer-0.0.1-SNAPSHOT.jar
```

> Точка входа в приложение: класс TgOrganizerApplication.java
